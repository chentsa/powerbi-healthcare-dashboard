# 🏥 Interactive Power BI Healthcare Dashboard

## 📌 Objective
Built an interactive Power BI dashboard for hospital management enabling 
real-time tracking of patient visits, instrument availability, revenue 
trends, and appointment waiting times.

## 🛠️ Tools Used
- Power BI (Dashboard & Data Modeling)
- Microsoft Excel (Data Preparation)

## 📊 Key Features
- Real-time tracking of patient visits and appointment wait times
- Instrument availability monitoring
- Revenue trend analysis
- Drilldown functionalities with KPI-driven insights
- Custom visuals with transformed datasets
